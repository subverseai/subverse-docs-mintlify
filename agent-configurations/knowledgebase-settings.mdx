Connect knowledge bases for information retrieval.

### Knowledge Base
Select from your created knowledge bases.

**Configuration:**
1. Create knowledge base in Integrations section
2. Select from dropdown
3. Agent can now query this information during calls

**Use Cases:**
- Product information
- Company policies
- FAQ answers
- Technical documentation

<Note>
  Knowledge bases must be created and ready (green status) before they can be used by agents. See [Knowledge Base Integration](/integrations/knowledge-base) for setup.
</Note>
